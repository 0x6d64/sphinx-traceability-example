The only purpose of the exemplum package is to demonstrate the functionality of the melexis fork of the sphinx-traceability-extension.

In this example we pretend to implement a simple piece of software. Its requirements are documented in the section Requirements.

The chapter Module documentation contains documentation of the implementation and the chapter Unittest documentation the documentation of the tests.

The chapter Version history contains an overview over versions of the example software.
